# Crimson Spider

Crimson Spider was built as a hackathon app. The project is a lightweight Angular-based prototype centered on a simple spider-graph chart experience.

## What Was Completed

- Built the initial hackathon prototype in Angular.
- Completed a simple spider-graph chart implementation.
- Reached a state worth preserving as an archived prototype rather than continuing active development.

## Project Status

This repository is archived and should be treated as a historical hackathon snapshot. It is not actively maintained.

## Security Notice

The app depends on outdated packages and currently has known security vulnerabilities in its dependency tree. Do not treat this project as production-ready without first upgrading dependencies and revalidating the application.

## Local Development

This project was originally generated with [Angular CLI](https://github.com/angular/angular-cli) version 1.7.4.

- Run `ng serve` for a dev server, then open `http://localhost:4200/`.
- Run `ng build` to create a build in `dist/`.
- Run `ng test` to execute unit tests via Karma.
- Run `ng e2e` to execute end-to-end tests via Protractor.
